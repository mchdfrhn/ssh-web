# Google Business Profile — Setup Guide untuk Surupan Software House

## Kenapa Penting?
- Muncul di Google Maps & Search ketika orang cari "jasa bikin website Jakarta"
- Bisa dapat Google Reviews = social proof gratis
- Gratis 100%, tapi impact besar buat SEO lokal

## Step-by-Step (15 menit)

### 1. Buka Google Business Profile
- Kunjungi: https://business.google.com
- Login dengan akun Google (pakai email info@surupan.tech kalau sudah ada)

### 2. Isi Data Bisnis
| Field | Isi |
|---|---|
| Nama bisnis | **Surupan Software House** |
| Kategori utama | **Software Company** |
| Kategori tambahan | Web Designer, Computer Consultant |
| Alamat | (isi alamat kantor/ko-working space) |
| Telepon | +62 889 7108 4208 |
| Website | https://surupan.tech |
| Jam buka | Senin-Jumat 09:00-18:00 |

### 3. Verifikasi
- Google kirim postcard ke alamat (7-14 hari) ATAU verifikasi via telepon
- Setelah verified, listing langsung live

### 4. Optimasi Setelah Live
- **Upload foto**: Logo, foto kantor/ko-working, screenshot portfolio
- **Tambah deskripsi**:

```
Surupan Software House adalah agensi digital yang membantu UMKM, startup, dan institusi pemerintah membangun website, aplikasi, dan sistem digital. Mulai dari landing page Rp 750rb hingga custom web app. WhatsApp-first approach, harga transparan, garansi 30 hari. Dipercaya oleh STTPU dan Pusdatin Kementerian PUPR.
```

- **Tambah layanan**:
  1. Landing Page — Mulai Rp 750rb
  2. Custom Web Application — Mulai Rp 3 Juta
  3. Maintenance & DevOps — Mulai Rp 500rb/bulan
  4. Paket Go Digital — Rp 2.5 Juta

### 5. Minta Reviews dari Klien
Setelah setiap project selesai, kirim link review ke klien:

```
Halo Kak [NAMA], website sudah live ya! 🎉

Kalau berkenan, boleh kasih review di Google? Ini link-nya:
https://g.page/r/[ID_BISNIS_ANDA]/review

Review dari Kak [NAMA] sangat berarti untuk bisnis kami 🙏
```

## Target: 5 reviews pertama dalam 1 bulan
- Review dari STTPU
- Review dari Pusdatin
- Review dari klien berikutnya
- 2 review dari rekan/partner
